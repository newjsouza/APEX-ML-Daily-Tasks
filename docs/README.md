# APEX Monitor v2.0

Plataforma profissional para monitoramento, análise e gestão de apostas esportivas.

## Funcionalidades
- Dashboard com métricas em tempo real
- Gestão de apostas, múltiplas e bingos
- Upload e histórico de imagens
- Sincronização automática com GitHub
- Relatórios diários e detalhamento de partidas
- Notícias e comunicados integrados
- Design modular e responsivo

## Estrutura do Projeto
```
├── backend/

│   └── README.md
└── .env
```

## Instalação e Execução
1. Instale as dependências:
   ```bash
   npm install
   ```
2. Configure o arquivo `.env` com suas credenciais do GitHub.
3. Inicie o servidor:
   ```bash
   npm start
   ```
